More About This Site

This website has been a dreamed about project for years now, and it's finally coming to fruition. I've been running off of tumblr so far, but it doesn't offer the creative outlet my brain needs for my passions. For a while now, I've been subscribed to The Marginalian (which if you haven't ever heard of this one-woman show, please go show her some support, she's amazing) and it's given me a ton of inspiration, pushing me to write and dive into my love for random fascinations.

I'm a massive lover of history, art, writing, and all things science. I'm currently in college for Software Development, and while I love coding in general - with a focus on Java - I absolutely love the creativity of web development. While I'm currently learning bootstrap as well for my side job, I'm challenging myself through this website to fully use only HTML, CSS, and eventually Javascript once it's better laid out.

Please feel free to go to my guestbook, I would love to hear from you all. Until then, explore, enjoy, and learn with me!
